## Requirements

- Node.js 20 or later
- TypeScript 5.x (recommended)

## Install

```bash
# npm
npm install crag

# pnpm
pnpm add crag

# yarn
yarn add crag

# bun
bun add crag
```

## JSR

crag is also available on [JSR](https://jsr.io/@bgub/crag):

```bash
npx jsr add @bgub/crag
```
