---

## Requirements

- Node.js 20 or later
- TypeScript 5.x (recommended)

## Install

```bash
# npm
npm install helm

# pnpm
pnpm add helm

# yarn
yarn add helm

# bun
bun add helm
```

## JSR

helm is also available on [JSR](https://jsr.io/@bgub/helm):

```bash
npx jsr add @bgub/helm
```
