---

## Requirements

- Node.js 20 or later
- TypeScript 5.x (recommended)

## Install

```bash
# npm
npm install bevel

# pnpm
pnpm add bevel

# yarn
yarn add bevel

# bun
bun add bevel
```

## JSR

bevel is also available on [JSR](https://jsr.io/@bgub/bevel):

```bash
npx jsr add @bgub/bevel
```
